代码基础知识

了解更多HTML+CSS基础课程
初识代码——html、css、js之间的关系
  • HTML是页面的内容
  • CSS就是负责页面样式的，美化HTML的
  • js(javascript)是让HTML更生动，更好看，让网页会动起来 不需要掌握

认识html文件（后缀为“.html”）:



<!DOCTYPE HTML>
<html>
	<head>
	</head>
	<body>
	</body>
</html>

认识css文件（后缀为“.css”）如下:


<style>
h1{font-size:14px;}
p{color:#252525;}
a{text-algin:center;}
</style>

认识js文件（后缀为“.js”）如下:


<script text="text/javascript">
document.getElementById("#cro");
$("#cro").css("color","red");
</script>
认识html标签

常用标签：了解更多标签

div、li、p、a、img、及h系列

标签的规则：一般标签都需要闭合

如下面： 

<!DOCTYPE HTML>
<html>
	<head>
	</head>
		<h1>这是一个主标题标签</h1>
		<h2>这是一个副标题标签</h2>
		<div>这是一个div标签</div>
		<ul>
			<li>这是一个无需列表标签</li>
			<li>这是一个无需列表标签2</li>
		</ul>
		<p>这是一个段落标签</p>
		<a href="#">这是一个超链接标签</a>
		<img src="" alt="这是一个图片标签">
	<body>
	</body>
</html>

重点认识两个标签：img标签 a标签


<a href="链接的地址" target="_blank" >这是一个超链接标签</a>
<img src="图片的地址" alt="这是一个图片标签">

关于a标签的属性: 了解更多a

  • href：规定链接指向的页面的 URL。
  • target：规定在何处打开链接文档。

    _blank:浏览器总在一个新打开、未命名的窗口中载入目标文档。

    _self:默认。在相同的框架中打开被链接文档。

    _parent: 在父框架集中打开被链接文档。

    _top:在整个窗口中打开被链接文档。

关于img标签的属性: 了解更多img

  • src：规定显示图像的 URL。
  • alt：规定图像的替代文本。
  • width：设置图像的宽度。
  • height：定义图像的高度。

专题构成

专题组成部分

  • html 编辑主要操作个更改文件，不要随意修改文件的名
  • images 图片的存放位置
  • css 样式文件的寻访位置
  • js js文件的存放位置

专题包至少含有一个"index.html",其他文件并不是必要的

专题上传注意事项
  • 至少包含"index.html"文件 专题最基础的组成部分
  • 专题上传压缩包格式必须是".zip" 后台上传只认这种模式
  • 专题标签必须都闭合 代码的基本规则

上传专题的技巧：如果出现代码未闭合的情况，你可只压缩"index.html"上传（节约上传时间）,后台检查通过后再整包上传

专题制作

海报专题制作

增加知识点：认识css:background

background:#000 url(images/bg1.jpg) no-repeat center 0;

  • #000: 背景颜色，16位色值
  • url(图片的地址): 背景图片的地址
  • no-repeat: 背景不重复
  • center 0：center表示水平居中，0表示居上0

海报专题包下载

  • 第一步：下载海报专题报;
  • 第二步：把切好的图片放入专题包里面的images文件夹中
  • 第三步：用小黑打开index.html文件，在文件中找到下面 
    
<!-- 海报内容区域开始 -->
<div class="part part1"></div>
<div class="part part2"></div>
<!-- 如需增加更多，以次增加下面代码，并以次修改后面的数值 -->
<!-- <div class="part part3"></div> -->
<!-- 海报内容区域结束 -->
    在此处增加相应代码与你的图片数相同; 比如：海报专题切了5张图对应如下 
    
<!-- 海报内容区域开始 -->

<div class="part part2"></div>
<div class="part part3"></div>
<div class="part part4"></div>
<div class="part part5"></div>

<!-- <div class="part part3"></div> -->
<!-- 海报内容区域结束 -->
  • 第四步：修改背景图片的地址和高度；
